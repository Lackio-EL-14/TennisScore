    - Probar secuencia 0-15-30-40-game para jugador 1 y 0 para jugador 2
    - Probar secuencia para ambos jugadores al mismo tiempo
    - Al llegar a "game" devolver los contadores a 0

    - si ambos jugadores logran punto el marcador volvera a tener "Deuce"
    - Si uno obtine 2 puntos mostrar game y devolver los marcadores a 0
